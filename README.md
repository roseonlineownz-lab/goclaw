## ✨ Features

- **Single Binary**: No dependencies
- **11+ LLM Providers**: OpenAI, Anthropic, Google, Ollama, etc.
- **5 Channels**: Route requests to different models
- **Teams & Delegation**: Multi-agent orchestration
- **MCP Compatible**: Works with Claude Code

---

## 🚀 Quick Start

```bash
# Build
go build -o goclaw ./cmd/goclaw

# Run
./goclaw --config config.yaml
```
